Nice Track

Nice Track is a prototype for a quantitative journaling application. Everyday, a user can rate topics of interest to their lives and add a short journal entry about the rating. As more ratings are added, new charts are revealed to provide deeper insight. I developed this on Meteor, a fullstack javascript framework for building apps.
